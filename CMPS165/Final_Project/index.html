California Wildfires 2008 to 2016

Purpose

The purpose of this visualization is to display statistics of California Wildfires from 2008 to 2016. Three kinds of data can be viewed which are the number of fires, how many acres were burned, and the cost of the damage done by the wildfires. The data can be filtered by cause and can be viewed by year.

Observations

We were unable to draw any conclusions because the data fluctuates unpredictably which makes it difficult to find any possile trends to determine conclusions.
